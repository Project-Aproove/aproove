Roadmap do Projeto

Da ideia ao produto — quatro fases para construir o diário do fundador mais usado no Brasil.

Fase 1 — MVP Web
Mar 2026
Concluída ✓
  • Autenticação segura (SHA-256 + token diário)
  • Auth persistente entre deploys via aba Config do Sheets
  • Admin master vitalício (MASTER_ADMIN hardcoded)
  • Captura e gerenciamento de ideias
  • Check-in inline (formulário aparece na página, sem popup)
  • Checkout de sessões de trabalho
  • Banco de dados Google Sheets com 3 abas (Ideas, Sessions, Config)
  • Deploy no Fly.io free tier (região GRU — não dorme)
  • Reset de senha por email (Gmail SMTP)
  • Interface multilingual (PT/EN/FR/ES/DE — 5 idiomas)
  • Vínculo visual ideia ↔ sessão
  • Geração de conteúdo para redes sociais via IA
  • Logout com username exibido no header
  • Aba App com prompt Stitch (visível só para logados)
Fase 2 — Síntese e Distribuição
Q2 2026
Em andamento
  • Síntese periódica com IA (Gemini free) — concatena ideias + sessões do período
  • Digest enviado por WhatsApp (Meta Business API — 1.000/mês grátis) e email
  • Configurações do usuário: período de síntese, tom de voz, WhatsApp
  • OAuth com Google e GitHub (substituir username/senha)
  • App mobile via Google Stitch (iOS + Android)
  • Push notifications para check-in diário
  • Widget de captura rápida na tela inicial
Fase 3 — Monetização
Q3 2026
Planejada
  • Publicação na App Store (iOS)
  • Publicação no Google Play (Android)
  • Free: síntese manual, ideias ilimitadas, sem WhatsApp
  • BBrain Pro: $4,99/mês — síntese automática + WhatsApp digest
  • BBrain Teams: $12/usuário/mês — ideias compartilhadas + dashboard
  • Integração de pagamento via Stripe
  • Domínio bbrain.app (~R$80/ano no Porkbun)
Fase 4 — Escala
Q4 2026
Planejada
  • Integração com WhatsApp (bot de captura)
  • IA avançada — análise de padrões de ideias
  • Insights semanais gerados por IA
  • Export em PDF (relatório de sessões + ideias)
  • API pública para desenvolvedores
  • Comunidade de fundadores BBrain
